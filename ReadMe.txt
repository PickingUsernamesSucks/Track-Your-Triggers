Track Your Triggers is designed to be an aid to the game Magic the Gathering.

It is intended to display the triggered abilities of a player's cards in an easy to find format. 

To use the app, simply press the "Start Game" Button on the Home screen. Then, as you play cards, type their name into the text box next to the "Add a card to the battlefield" text. Then hit enter or the Search button.
Note - the search function can only return 1 card. So you must give it enough information to determine which card you have. You do not need to type out the exact name of the card. But, if you play someone like Elspeth who has several versions, input more of that particular card's name. The search can account for some misspellings, but you have the card in front of you, right?

As you add cards to the battlefield, the right side of the screen will automatically fill with those card's triggers, if any. The triggers will be grouped by when they happen.

You can also remove cards from the battlefield, which will remove their triggers as well. 

The app does show active and static abilities via showing a cards entire text as a nice boon, but these abilities are not given extra attention like triggered abilities.

This app is not an online way to play mtg, nor is it a paid product.

For an online way to play mtg, see Magic the Gathering Arena or Magic the Gathering Online.
